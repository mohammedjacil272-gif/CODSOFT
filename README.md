# Iris Flower Classification

This project is done as part of the CodSoft Data Science Internship.

## Project Objective

The main objective of this project is to classify iris flowers into different species using Machine Learning.

The three species are:
- Setosa
- Versicolor
- Virginica

## Technologies Used

- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm

K-Nearest Neighbors (KNN)

## Steps Performed

1. Imported required libraries
2. Loaded iris dataset
3. Checked dataset information
4. Visualized the data using graphs
5. Split dataset into training and testing data
6. Trained KNN model
7. Predicted flower species
8. Calculated model accuracy

## Output

The model predicts the flower species with good accuracy.
